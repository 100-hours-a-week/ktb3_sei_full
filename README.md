# shhh..(쉿)
>우리의 하루가 어떤 모양이어도  
쉿 - 여기서는 괜찮아

shhh..(쉿)은 오늘 하루 있었던 일을 공유할 수 있는 **비밀 일기장**입니다.

남들에게 말하지 못했던 일들,   
부끄러운 실수,   
혼자만의 고민을 공유하고 위로와 공감을 얻을 수 있어요.
